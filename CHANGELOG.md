# 更新日志 / Changelog

## v1.0.2 (2025-01-04)

### 🎯 追踪指南针系统改进

#### 新增功能
- ✨ 改进追踪指南针交互方式：按Q键（丢弃物品键）循环切换追踪目标
- 🔒 禁止猎人丢弃追踪指南针，防止误操作
- 🎯 优化追踪体验，更符合原版Dream猎人追逃玩法

#### Bug修复
- 🐛 修复背包满时按Q键导致指南针消失的严重bug
- 🐛 修复指南针可能被放置到装备栏或副手的问题
- 🐛 修复通过反复按Q键刷物品的漏洞

#### 优化改进
- ⚡ 优化指南针放回逻辑：
  - 优先在快捷栏（slot 0-8）查找空位
  - 其次在主背包（slot 9-35）查找空位
  - 背包满时自动丢弃快捷栏第一格物品，确保指南针不会消失
  - 不会将指南针放置到装备栏或副手
- 🎮 允许玩家在背包中自由移动指南针
- 🔄 保持每秒自动更新目标位置的实时追踪功能

### 📋 赛季系统配置

#### 新增功能
- 📁 添加 `season.yml` 配置文件模板
- 🔧 优化赛季配置加载逻辑，使用 `saveResource()` 方法
- ⏰ 首次运行时自动设置赛季开始时间

#### 配置说明
```yaml
# 赛季持续天数
season-duration-days: 30  # 30天后自动开始新赛季
                          # 设置为0表示无限期赛季
```
